mode

Website mode

Fetch HTML → extract “article-ish” content → normalize to clean text. If extraction looks blocked or too thin, retry via Firecrawl Markdown (optional). With --format md, the CLI prefers Firecrawl Markdown when configured and can also convert HTML → Markdown via --markdown-mode (LLM) or uvx markitdown.

Podcast pages are supported (Apple Podcasts, Spotify episodes, RSS feeds, Podbean, Podchaser, embedded YouTube). Summaries use published transcripts when available, otherwise Whisper.

Flags

  • --firecrawl off|auto|always
  • --timeout 30s|2m|5000ms (default 2m)
  • --extract (print extracted content; no summary call)
  • --format md|text (default text)
  • --markdown-mode off|auto|llm|readability (only with --format md)
  • --preprocess off|auto|always (default auto; controls markitdown usage)
  • --json (emit a single JSON object)
  • --verbose (progress + which extractor was used)
  • --metrics off|on|detailed
Plain-text mode: --extract --format text.
